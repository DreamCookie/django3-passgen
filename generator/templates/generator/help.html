What this service about:

What we have done is called the Linear Congruent Method.
It is very commonly used because it is easy to implement and calculations are fast.
In different programming languages, the implementation of the linear congruent method is different,
that is, the values of the constants change.
For example, the random number function in libc (the standard C library for Linux) uses m = 2^32, a = 1664525, and c = 1013904223. Compilers like gcc usually use these values.
